Location object has a reload() method using which you can reload the current web page:- location.reload();
  • If you pass true to the reload() method, then it will force the browser to reload the web page from the server.
  • location.reload(true);
  • On the other hand, if you pass false or don't pass anything to the reload() method, the browser might reload the page from the cache.
