Conclusion

Event listeners and event handlers are used to handle events, such as button clicks on your page. There is a subtle difference between event listeners and event handlers: An object can only have one event handler for a specific event but it can have multiple event listeners for the event. When in doubt, prefer event listeners over event handlers.
