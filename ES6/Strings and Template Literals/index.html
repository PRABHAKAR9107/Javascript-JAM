Different String Methods
  • String.startsWith(substring,startpoint):-return true if string starts with given substring
  • String.endsWith(substring,endIndex):-return true if string ends with given substring
  • String.includes(substring,position):-return true if string including given substring
  • string.repeat(count):-The repeat() method returns a string with a number of copies of a string.
